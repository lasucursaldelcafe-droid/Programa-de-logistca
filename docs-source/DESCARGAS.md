## Una sola app, sincronizada

Web, Windows, Android y Linux comparten **el mismo backend Firebase**. Inicia sesión con la misma cuenta en cualquier plataforma; eventos, personal, turnos y asistencia se actualizan en tiempo real.

| Plataforma | Archivo |
|------------|---------|
| **Web** | https://lasucursaldelcafe-droid.github.io/Programa-de-logistca/login |
| **Windows** | `SPE-Admin-*-nsis.exe` o `*-portable.exe` |
| **Linux** | `SPE-Eventos-*-linux.AppImage` |
| **Android** | `SPE-Eventos-*-android.apk` |
| **iOS** | Safari → Añadir a pantalla de inicio (PWA) |

Releases: https://github.com/lasucursaldelcafe-droid/Programa-de-logistca/releases/latest
